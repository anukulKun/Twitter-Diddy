# Twitter Diddy
A simple Twitter scraper using Nitter, built to fetch tweets, hashtags, and user timelines without using the official Twitter API.

### Tech Stack
- Python  
- ntscraper (Nitter API Wrapper)  

### Run Locally

```bash
# Clone the repo
git clone https://github.com/anukulKun/Twitter-Diddy.git

cd Twitter-Diddy

# Install dependencies
pip install -r requirements.txt

# Run the notebook or script
jupyter notebook
# or
python main.py
